Includes social sharing buttons before/after post content.

== Description ==

Social Buttons is Just Another Social Plugin, as it doesn't do any special, we can say its main feature is clean code.

Buttons currently supported:

* Facebook like
* Tweeter Share
* Google Plus One

Social Buttons requires PHP 5.2.


== Installation ==


1. Upload the `social_buttons` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure as needed
